# v4 / v4b — ranked-only data + context + flip aug + decode wins

**Purpose:** the v4 batch — switching to ranked-only data, more context (crop 4096 / attn-levels 3),
h/v flip augmentation, and the decode/post-process wins that still ship today. | **STATIC** (frozen).

Run detail + play feedback: [status/results.md](../status/results.md). (Old RESEARCH §10.0–10.1.)

## v4 — full curated library (previous release)

31,270 curated maps (≤12★), base 128. Run killed by an OS sleep at e16 (undertrained but strong, loss
0.0077). SR monotonic, 16–17/19 metrics in-range. Superseded by v4b (ranked data fixes the junk that
≤12★ curation missed).

## v4b — ranked train (the release)

Ranked-only data (`osu!.db` filter → ~23.6k ranked/approved/loved maps), **more context** (`--crop
4096 --attn-levels 3`), **h/v flip augmentation**, train/val split. epoch 48, val 0.00486 (well below
v4's 0.0077).

**Eval:** SR monotonic; 17–19/19 metrics in-range (beats v4's 16–17); hitsounds ≈ real. SR offset
persists → use `--match-sr`.

**Play feedback (v4b vs v4):**
- ✅ Kiai much better (sections start near-perfect; v4 lagged ~10–12 s); minor: ends 1–3 s early.
- ✅ No dead trailing note; hitsounds slightly better.
- ✅ Jumps / patterns / streams much better — **validates ranked data + context + flip aug**.
- ⚠️ Rhythm REGRESSED vs v4: strange 0.5–2 s pauses; some notes off the ¼ grid (look 1/6 or 1/8).
- ➖ No spinners; curve sliders still low (the slider-representation gap → [v5.md](v5.md)).

## Decode / post-process wins (shipped on v4, still in the code)

- `clamp_slider_endpoints` — caps slider length so osu! extrapolation can't shoot tails off the
  playfield (0/116 off-field after `snap_slider_ends`).
- `decode_signal(accent_threshold=0.85)` — accent channels saturate near +1; 0.85 → ~0.33 hitsound
  usage (matches real).
- `trim_isolated_ends` — asymmetric trailing trim (2.2 s) + drops a lone circle after the final
  spinner (phantom spin-down note).
- `snap_to_grid` loosened 45→60 ms / 40→50% — *suspected contributor to the v4b rhythm regression;
  revisited at v5* ([v5.md](v5.md)).
- `compute_breaks` + `write_osu(breaks=)` — `[Events]` breaks for gaps ≥3.5 s (cosmetic; marks
  existing gaps only).
- `--match-sr` — iterate to hit the exact target rating.

## v4b play-feedback action items (the v5 backlog)

- **Rhythm regression** (top decode priority, no retrain): test reverting snap 60→45 ms; add 1/6, 1/8

- **Kiai ends 1–3 s early** — extend decoded kiai end to the next downbeat.
- **No spinners** — `spinner_hold` rarely fires at decode; low priority.
- **Curve sliders / streams** — the representation fixes (v5 slider channels for curves+reverse; flow
