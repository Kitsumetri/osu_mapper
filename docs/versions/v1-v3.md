## What v3 established (the 10-channel signal)

The v3 signal set — `onset, slider_hold, spinner_hold, new_combo, cursor_x, cursor_y, kiai_hold,
whistle, finish, clap` (channels 0–9, still the base of every later version) — plus:
- **Difficulty conditioning**: a 6-D context vector `[SR, AR, OD, HP, CS, density]` embedded and added
  to the timestep embedding (FiLM path). SR = exact rosu-pp value.
- **Classifier-free guidance (CFG)**: train with the context dropped to a learned null ~15% of the
  time; sample with guidance weight `w` to push toward the requested difficulty. Without CFG a weak
  scalar condition is ignored.

These shipped and are still the conditioning mechanism (math in
[knowledge/diffusion-math.md](../knowledge/diffusion-math.md) §6.4).

## Known v3-era gaps (carried forward)

- Single-line / straight sliders — the slider shape was traced into the shared cursor channels and
  denoised to near-straight. Fixed at **v5** with dedicated anchor channels ([v5.md](v5.md)).
- No reverse sliders (`slides` dropped) — also fixed at v5.
- No SV — added as a learned channel at **v7** ([v7.md](v7.md)).
- "Random clusters" / weak patterns — the under-dispersion problem that drives v7→v9
  ([knowledge/lessons-learned.md](../knowledge/lessons-learned.md)).
- base-160 + bf16 divergence under ε-prediction (v2 @e21, v3 @e12) — unblocked at **v8** by v-pred +
  zero-terminal-SNR.
